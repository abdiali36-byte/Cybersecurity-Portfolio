# Cybersecurity Portfolio

This portfolio showcases defensive security skills, SOC investigations, incident response documentation, cloud security case studies, automation scripts, and ongoing learning progress aligned to Blue Team and SOC Analyst career pathways.

The objective of this portfolio is to demonstrate:
- Ability to investigate alerts based on real-world attack behaviours
- Understanding of identity, access and cloud-based security controls
- Log analysis, dashboarding and detection engineering concepts
- Documentation and communication of findings in a structured format
- Continuous progression through hands-on projects and certifications

---

## Repository Structure

| Folder | Purpose |
|--------|---------|
| SOC-Investigations | Case studies and investigations based on simulated attacks and SIEM alerts |
| Incident-Reports | Formal writeups following incident response frameworks |
| TryHackMe-Writeups | Writeups for selected rooms with defensive relevance |
| SIEM-Dashboarding-and-Detection | Log queries, dashboards and alert logic |
| Network-Security | Packet analysis, firewall rules, and protocol behaviour |
| Identity-and-Access | MFA, SSO, Conditional Access, and account security |
| Cloud-Security | Investigations and configurations in cloud environments |
| Security-Automation-and-Scripting | Python/PowerShell scripts to support SOC tasks |
| Threat-Intelligence | IOC research, OSINT enrichment and threat context |
| SecurityPlus-Notes | Study notes and revision summaries |
| Roadmap-Progress | Weekly reflection and roadmap alignment |
